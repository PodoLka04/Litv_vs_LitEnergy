# Литвин битва за Кондиции

Проект создан для публикации на платформе **Yandex Games** с целью коммерции.

**Платформа:** WebGL (для ПК и мобильных устройств)

## 🎮 О проекте

**Литвин Битва за Кондиции** — уникальный кликер, где твоя цель: кликать по Литвину, зарабатывать валюту, прокачивать силу через выкладывание вирусных видео и побеждать боссов, чтобы получить легендарные энергетики LitEnergy.

Стань настоящим чемпионом, открывай новые уровни, скины и апгрейды.

## 🎯 Основные механики

- **Кликер:** Тапай по Литвину — каждый клик приносит валюту.
- **Прокачка:** Улучшай силу клика, выкладывая вирусные видео.
- **Боссы:** Сражайся с боссами, получай энергетики LitEnergy.
- **Развитие:** Разблокируй новые уровни, скины и апгрейды.

## 🚀 Сборка и запуск

1. Скачайте репозиторий.
2. Откройте проект в Unity (версию смотрите в `LITV+KVADR/ProjectSettings/ProjectVersion.txt`).
3. Для запуска в редакторе откройте сцену `LITV+KVADR/Assets/Scenes/MainScene.unity` и нажмите `Play`.
4. Для публикации на Yandex Games соберите проект в **WebGL** через `File → Build Settings`.
5. Загрузите сборку в личном кабинете разработчика Yandex Games.

# Litvin Battle for Konditsii

The project was created for the **Yandex Games** platform in profit.

**Platform:** WebGL (for PC and mobile devices)

## 🎮 About the project

**Litvin Battle for Konditsii** is a unique clicker game where you tap on Litvin, earn currency, upgrade your power by posting viral videos, and defeat bosses to get legendary LitEnergy drinks.

Become a true champion, unlock new levels, skins, and upgrades.

## 🎯 Core mechanics

- **Clicker:** Tap on Litvin — each click earns currency.
- **Upgrades:** Improve click power by posting viral videos.
- **Bosses:** Fight bosses to earn LitEnergy drinks.
- **Progression:** Unlock new levels, skins, and upgrades.
